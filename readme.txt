== Description ==

Tema Profissional, Moderno e Poderoso para Seu Site em WordPress. 

== Installation ==

A partir do seu painel Wordpress:

1. **Acesse** Temas > Adicionar Novo
2. **Clique** em 'Enviar tema'
3. **Selecione** o arquivo .ZIP que você baixou.
4. **Clique** em 'Instalar agora'
5. **Clique** em 'Ativar'
